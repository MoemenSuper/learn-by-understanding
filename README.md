## What It Does

This skill teaches any topic by starting with what each word, symbol, command, or step literally means.

It pushes the teacher to explain:

- where each thing comes from
- what it does
- what it returns or changes
- why that method was chosen
- what breaks without it
- how to test the learner's understanding

## Teaching Loop

1. Explain the literal meaning.
2. Show a tiny example.
3. Connect it to the real task.
4. Ask the learner to predict.
5. Let the learner try.
6. Give hints before the answer.
7. End with "explain it back."

## Use It For

- programming lessons
- math or science concepts
- tool workflows
- project walkthroughs
- any explanation that feels too abstract
